¿Qué es CSS3?

El lenguaje del diseño y la presentación web

CSS (Cascading Style Sheets — Hojas de Estilo en Cascada) es el lenguaje que controla la apariencia visual de las páginas web. Mientras HTML define qué hay en la página, CSS define cómo se ve: colores, tamaños, posiciones, fuentes y animaciones.

CSS3 es la versión actual, que incorporó módulos independientes como Flexbox, Grid, animaciones, transformaciones, variables personalizadas y soporte avanzado para tipografía y colores.

💡

Analogía: Si HTML es el esqueleto de una persona, CSS es la ropa, el maquillaje, el color de cabello y el estilo. Puedes cambiar completamente la apariencia sin tocar la estructura interior.

🎨 Presentación
Controla colores, fondos, bordes, sombras y todo lo visual de cada elemento HTML.
📐 Diseño
Define cómo se organizan los elementos en la página: columnas, filas, posicionamiento.
📱 Responsivo
Adapta el diseño a diferentes tamaños de pantalla con Media Queries.
✨ Animaciones
Crea transiciones suaves, animaciones y transformaciones sin JavaScript.
3 formas de usar CSS 
/* ─── 1. CSS EXTERNO (recomendado) ─── */
/* En el <head> del HTML: */
<link rel="stylesheet" href="estilos.css"/>

/* ─── 2. CSS INTERNO (dentro del HTML) ─── */
<style>
  body { background: #0f0f1a; }
</style>

/* ─── 3. CSS EN LÍNEA (evitar para mantenimiento) ─── */
<p style="color: pink; font-size: 1.2rem;">Texto</p>

Sintaxis y Variables

La anatomía de una regla CSS

Una regla CSS tiene dos partes: el selector (qué elementos afecta) y el bloque de declaraciones (cómo deben verse). Cada declaración tiene una propiedad y un valor separados por dos puntos.

sintaxis.css 
/* ── ANATOMÍA DE UNA REGLA ── */
/*  selector  { propiedad: valor; }  */

h1 {
  color: #ec4899;          /* propiedad: valor */
  font-size: 3rem;
  text-align: center;
}

/* ── VARIABLES CSS PERSONALIZADAS (Custom Properties) ── */
:root {
  --color-primario:    #ec4899;
  --color-secundario: #8b5cf6;
  --fuente-principal: 'Montserrat', sans-serif;
  --radio-borde:       12px;
  --sombra:            0 4px 20px rgba(0,0,0,.3);
}

/* ── USO DE VARIABLES ── */
.tarjeta {
  background:    var(--color-primario);
  border-radius: var(--radio-borde);
  box-shadow:    var(--sombra);
  font-family:   var(--fuente-principal);
}

/* ── COMENTARIOS CSS ── */
/* Este es un comentario de una línea */
/*
  Este es un comentario
  de varias líneas
*/

/* ── UNIDADES DE MEDIDA ── */
.ejemplo {
  width:      300px;    /* píxeles — absoluto */
  font-size:  1.5rem;  /* relativo al root (16px) */
  padding:    2em;     /* relativo al font-size del elemento */
  height:     50vh;    /* 50% del alto de la ventana */
  max-width:  80%;     /* porcentaje del contenedor padre */
  gap:        clamp(1rem, 3vw, 2rem); /* mín, ideal, máx */
}
px
Píxeles. Unidad absoluta. 1px = 1 punto en pantalla.
font-size: 16px
rem
Relativo al font-size de :root (normalmente 16px). El más recomendado.
font-size: 1.5rem → 24px
em
Relativo al font-size del elemento padre. Se acumula en anidamiento.
padding: 2em
vh / vw
Viewport Height / Width. 100vh = toda la pantalla de alto.
height: 100vh
%
Porcentaje del elemento padre. width: 50% = mitad del contenedor.
width: 50%
clamp()
Valor fluido entre un mínimo y máximo. Ideal para tipografía responsiva.
font-size: clamp(1rem, 4vw, 3rem)

Selectores y Especificidad

Apuntar exactamente al elemento correcto

Los selectores son patrones que indican a CSS qué elementos HTML deben ser estilizados. CSS3 ofrece una enorme variedad de selectores para apuntar con precisión quirúrgica a cualquier elemento.

Cuando dos reglas aplican al mismo elemento, la especificidad determina cuál gana. Una regla más específica siempre supera a una menos específica.

selectores.css 
/* ── SELECTORES BÁSICOS ── */
*          { box-sizing: border-box; }  /* universal */
p          { color: white; }             /* etiqueta */
.clase     { font-weight: 700; }        /* clase */
#id        { font-size: 2rem; }         /* id (único) */

/* ── SELECTORES COMBINADOS ── */
nav a      { color: #60a5fa; }   /* descendiente */
nav > a    { font-weight: bold; } /* hijo directo */
h2 + p     { margin-top: 0; }    /* hermano adyacente */
h2 ~ p     { color: gray; }      /* todos los hermanos siguientes */
h1, h2, h3 { font-family: serif; } /* grupo */

/* ── PSEUDO-CLASES (estado del elemento) ── */
a:hover              { color: #ec4899; }
button:focus         { outline: 2px solid #8b5cf6; }
input:disabled       { opacity: .4; }
li:first-child       { font-weight: bold; }
li:last-child        { border-bottom: none; }
li:nth-child(2n)     { background: rgba(255,255,255,.05); }
p:not(.especial)     { font-size: .9rem; }

/* ── PSEUDO-ELEMENTOS (partes del elemento) ── */
p::first-line   { font-variant: small-caps; }
p::first-letter { font-size: 3em; float: left; }
.btn::before    { content: '→ '; }
.btn::after     { content: ''; display: block; }

/* ── SELECTORES DE ATRIBUTO ── */
input[type="email"]    { border-color: #0ea5e9; }
a[href^="https"]       { color: #22c55e; }  /* empieza con */
a[href$=".pdf"]        { color: #f59e0b; }  /* termina con */
a[href*="google"]      { font-style: italic; } /* contiene */

La especificidad determina qué regla "gana" cuando varias apuntan al mismo elemento. Funciona como un sistema de puntos:

!important Fuerza máxima — evitar su uso
id → #elemento 100 puntos
.clase :pseudo-clase [atributo] 10 puntos
etiqueta ::pseudo-elemento 1 punto
* universal / herencia 0 puntos
⚠️

!important anula toda la cascada y dificulta el mantenimiento del código. Úsalo solo como último recurso cuando no puedas aumentar la especificidad de otra forma.

El Box Model

Cómo CSS calcula el espacio de cada elemento

Cada elemento HTML es una caja rectangular. El Box Model describe las capas de esa caja: contenido, relleno (padding), borde (border) y margen (margin). Entender esto es esencial para controlar el layout.

MARGIN — Espacio exterior (transparente)
BORDER — Borde visible del elemento
PADDING — Espacio interior
CONTENT — Texto / Imágenes
box-model.css 
/* ── PROPIEDADES DEL BOX MODEL ── */
.caja {
  /* CONTENIDO */
  width:  300px;
  height: 200px;

  /* PADDING (relleno interior) */
  padding: 20px;              /* todos lados */
  padding: 10px 20px;        /* vertical | horizontal */
  padding: 10px 20px 15px 5px; /* top right bottom left */

  /* BORDER */
  border:        2px solid #ec4899;
  border-radius: 12px;  /* esquinas redondeadas */
  border-top:    4px solid #8b5cf6; /* lado específico */

  /* MARGIN (espacio exterior) */
  margin: 20px auto;  /* centrado horizontal */
  margin-top: 40px;

  /* ¡MUY IMPORTANTE! box-sizing */
  box-sizing: border-box;
  /* Con border-box: width INCLUYE padding y border */
  /* Sin él: width SOLO es el contenido (confuso) */
}

/* Aplicar border-box a todo (buena práctica global) */
* { box-sizing: border-box; }

/* ── DISPLAY ── */
.bloque   { display: block; }    /* ocupa toda la línea */
.en-linea { display: inline; }   /* fluye con el texto */
.mixto    { display: inline-block; } /* en línea con dimensiones */
.oculto   { display: none; }     /* desaparece del flujo */

Colores, Fondos y Sombras

El universo visual de CSS3

CSS3 ofrece múltiples formas de expresar colores y una potente colección de propiedades para fondos y sombras. Los gradientes y el sistema HSL son herramientas particularmente útiles para diseño profesional.

colores-fondos.css 
/* ── FORMATOS DE COLOR ── */
.colores {
  color: #ec4899;              /* hexadecimal */
  color: rgb(236, 72, 153);     /* rojo, verde, azul (0-255) */
  color: rgba(236,72,153,.5);  /* + opacidad (0-1) */
  color: hsl(328, 81%, 60%);   /* tono, saturación, luminosidad */
  color: hsla(328,81%,60%,.8);/* + opacidad */
}

/* ── GRADIENTES ── */
.grad-lineal {
  background: linear-gradient(135deg, #ec4899, #8b5cf6, #0ea5e9);
}
.grad-radial {
  background: radial-gradient(circle at center, #f59e0b 0%, #ec4899 100%);
}
.grad-animado {
  background: linear-gradient(270deg, #ec4899, #8b5cf6, #0ea5e9);
  background-size: 400% 400%;
  animation: gradMove 4s ease infinite;
}

/* ── SOMBRAS ── */
.sombra-caja   { box-shadow: 0 4px 20px rgba(0,0,0,.3); }
.sombra-neon   { box-shadow: 0 0 20px #ec4899, 0 0 60px rgba(236,72,153,.3); }
.sombra-interna{ box-shadow: inset 0 0 15px rgba(0,0,0,.5); }
.sombra-texto  { text-shadow: 0 2px 10px rgba(236,72,153,.6); }

/* ── FONDO CON IMAGEN ── */
.hero {
  background-image:    url('imagen.jpg');
  background-size:     cover;      /* cubre todo el elemento */
  background-position: center;
  background-repeat:   no-repeat;
  background-attachment:fixed;     /* efecto parallax */
}

Tipografía

Fuentes, tamaños, espaciados y efectos

tipografia.css 
/* ── FUENTES EXTERNAS (Google Fonts) ── */
@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;700&display=swap');

/* ── FUENTE PERSONALIZADA (archivo local) ── */
@font-face {
  font-family: 'MiFuente';
  src: url('fonts/mifuente.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
}

/* ── PROPIEDADES DE FUENTE ── */
p {
  font-family:   'Montserrat', sans-serif; /* con fallback */
  font-size:     1rem;
  font-weight:   400;        /* 100-900 */
  font-style:    italic;
  line-height:   1.7;        /* espaciado entre líneas */
  letter-spacing:.05em;     /* espacio entre letras */
  word-spacing:  .1em;       /* espacio entre palabras */
}

/* ── ALINEACIÓN Y TRANSFORMACIÓN ── */
.titulo {
  text-align:      center;          /* left|right|center|justify */
  text-transform:  uppercase;       /* uppercase|lowercase|capitalize */
  text-decoration: none;            /* none|underline|line-through */
  text-indent:     2em;             /* sangría */
  white-space:     nowrap;          /* no dividir en líneas */
  overflow:        hidden;
  text-overflow:   ellipsis;        /* texto largo → "..." */
}

/* ── GRADIENTE EN TEXTO (CSS3 moderno) ── */
.texto-degradado {
  background: linear-gradient(135deg, #ec4899, #8b5cf6);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
font-family
Define la fuente. Siempre incluir fuente de respaldo: 'Fuente', sans-serif.
font-size
Tamaño del texto. Preferir rem o em sobre px para accesibilidad.
font-weight
Grosor: 100 (delgado) a 900 (negro). 400 = normal, 700 = negrita.
line-height
Altura de línea. 1.5 a 1.7 es ideal para lectura cómoda.
letter-spacing
Espacio entre caracteres. Valores positivos separan, negativos juntan.
text-shadow
Sombra del texto: offset-x offset-y blur color. Se pueden apilar varias.

Flexbox

El sistema de layout unidimensional

Flexbox es el sistema de layout de CSS3 para organizar elementos en una dimensión (fila o columna). Resuelve de forma elegante la alineación, distribución y ordenamiento de elementos que antes requerían hacks complejos.

justify-content: space-between
item 1
item 2
item 3
align-items: center + flex: 1
fijo
flex: 1 (crece)
fijo
justify-content: center
centrado
flexbox.css 
/* ── CONTENEDOR FLEX ── */
.contenedor {
  display:         flex;
  flex-direction:  row;         /* row|row-reverse|column|column-reverse */
  flex-wrap:       wrap;        /* nowrap|wrap|wrap-reverse */
  justify-content: space-between; /* eje principal */
  /* flex-start|flex-end|center|space-between|space-around|space-evenly */
  align-items:     center;      /* eje secundario */
  /* stretch|flex-start|flex-end|center|baseline */
  align-content:   flex-start;  /* múltiples líneas */
  gap:             16px;        /* espacio entre elementos */
}

/* ── ELEMENTOS FLEX ── */
.item {
  flex:        1;      /* crece para ocupar espacio disponible */
  flex-grow:   1;      /* capacidad de crecer */
  flex-shrink: 0;      /* no se encoge */
  flex-basis:  200px; /* tamaño base antes de crecer/encoger */
  align-self:  flex-end; /* sobrescribe align-items del padre */
  order:       2;      /* orden visual (default 0) */
}

/* ── PATRÓN CENTRADO PERFECTO ── */
.centrar {
  display:         flex;
  align-items:     center;
  justify-content: center;
  /* ¡El truco del millón! ↑ */
}

CSS Grid

El sistema de layout bidimensional

CSS Grid es el sistema más poderoso para diseño web. A diferencia de Flexbox (1D), Grid trabaja en dos dimensiones simultáneamente: filas y columnas. Es ideal para layouts completos de páginas.

grid-template-columns: repeat(3, 1fr) con span
Header (span 3)
Sidebar
Main (span 2)
Footer (span 3)
css-grid.css 
/* ── CONTENEDOR GRID ── */
.grid {
  display: grid;

  /* 3 columnas iguales */
  grid-template-columns: repeat(3, 1fr);

  /* Columnas con tamaños distintos */
  grid-template-columns: 250px 1fr 200px;

  /* Auto-fill responsivo sin media queries */
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

  grid-template-rows: auto 1fr auto;
  gap: 20px;          /* espacio entre filas y columnas */
  row-gap: 20px;      /* solo filas */
  column-gap: 16px;  /* solo columnas */
}

/* ── ELEMENTOS GRID ── */
.header {
  grid-column: 1 / -1;    /* de columna 1 a la última */
  grid-column: span 3;    /* ocupa 3 columnas */
}
.sidebar {
  grid-row: 2 / 4;         /* de fila 2 a la 4 */
}

/* ── LAYOUT COMPLETO CON NAMED AREAS ── */
.pagina {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main  "
    "footer  footer";
  grid-template-columns: 240px 1fr;
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

Transiciones, Transformaciones y Animaciones

CSS3 le da vida a la interfaz

CSS3 permite crear movimiento sin una sola línea de JavaScript. Las transiciones interpolan entre dos estados, las transformaciones mueven y deforman elementos, y las animaciones crean secuencias completas con keyframes.

TRANSITION (hover)
ROTATE
BOUNCE
SCALE PULSE
animaciones.css 
/* ── TRANSICIONES ── */
.boton {
  background: #ec4899;
  transition: all .3s ease;       /* propiedad duración timing */
  transition: background .2s ease, transform .3s cubic-bezier(.34,1.56,.64,1);
}
.boton:hover {
  background: #8b5cf6;
  transform: translateY(-4px);
}

/* ── TRANSFORMACIONES ── */
.transformar {
  transform: translateX(20px);        /* mover en X */
  transform: translateY(-10px);       /* mover en Y */
  transform: scale(1.2);              /* escalar 120% */
  transform: rotate(45deg);           /* rotar 45 grados */
  transform: skew(10deg, 5deg);       /* sesgar */
  transform: translate(10px,-5px) scale(1.1) rotate(3deg); /* combinar */
  transform-origin: center center;   /* punto de origen */
}

/* ── ANIMACIONES CON KEYFRAMES ── */
@keyframes flotarArriba {
  0%   { transform: translateY(0);     opacity: 1; }
  50%  { transform: translateY(-15px); opacity: .8; }
  100% { transform: translateY(0);     opacity: 1; }
}

.elemento {
  animation-name:            flotarArriba;
  animation-duration:        2s;
  animation-timing-function: ease-in-out;
  animation-delay:           .5s;
  animation-iteration-count: infinite;
  animation-direction:       alternate;
  animation-fill-mode:       forwards;
  /* Shorthand: */
  animation: flotarArriba 2s ease-in-out .5s infinite alternate;
}

/* ── MEDIA QUERIES (diseño responsivo) ── */
@media (max-width: 768px) {
  .grid { grid-template-columns: 1fr; }
  .titulo { font-size: 2rem; }
}
@media (prefers-color-scheme: dark) {
  :root { --bg: #0f0f1a; --text: #fff; }
}
