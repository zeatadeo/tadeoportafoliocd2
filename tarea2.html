SQL / MySQL
SQL

Structured Query Language. Lenguaje universal para gestionar bases de datos relacionales: consultar, insertar, actualizar y eliminar datos.

consulta.sql 
-- Consulta con JOIN y filtros
SELECT
  u.nombre,
  COUNT(p.id) AS total_pedidos
FROM usuarios u
JOIN pedidos p ON u.id = p.usuario_id
WHERE u.activo = 1
GROUP BY u.nombre
ORDER BY total_pedidos DESC;

Python 3.12
Python

El lenguaje más popular del mundo. Usado en IA, ciencia de datos, automatización, backends y aplicaciones de escritorio con Tkinter o PyQt.

main.py 
# Clase con dataclass y list comprehension
from dataclasses import dataclass

@dataclass
class Alumno:
    nombre: str
    calificacion: float

alumnos = [Alumno("Ana", 9.5), Alumno("Luis", 8.0)]
aprobados = [a for a in alumnos if a.calificacion >= 7]
print(f"Aprobados: {len(aprobados)}")
